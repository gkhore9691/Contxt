# Contxt — AI Context Pack Platform

Structured AI context packs for coding agents like Claude Code, Cursor, and Copilot.
Stop re-explaining your stack on every session — load a pack and start building immediately.

🔗 **Live:** Coming soon | 🐦 **Status:** Active development

## 🧠 The Problem

Every time you start a new AI coding session, you re-explain your stack, architecture, conventions, and preferences from scratch. Coldstart fixes this with portable, shareable context packs.

## ✨ Features

- Create and manage structured context packs (stack, architecture, conventions, preferences)
- Pack Health Score — rates how complete and useful your pack is
- AI Pack Optimizer — suggests improvements to your context
- GitHub README import — auto-generate a pack from your repo
- One-click copy for Claude Code, Cursor, and other agents
- CLI tool + VS Code extension (in progress)
- MCP Server integration (planned)

## 🛠 Tech Stack

Next.js · TypeScript · Supabase · Tailwind CSS · shadcn/ui

## 🚀 Getting Started
```bash
npm install
npm run dev
```
